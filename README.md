# CTF 

## Requirements
- Python 3.9 or above
- GNU Make

## Setup
```
git clone git@github.com:Zenjjim/CTF.git
cd CTF
make init
```

We have now created a db with some example Challenges and temporary user/password: `User1|test`

To run system
```
make run
```
